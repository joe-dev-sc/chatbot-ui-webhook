Installation

Add the following code to your HTML file:

<link href="https://your-domain.github.io/webhook-chatbot/assets/chatbot-widget.css" rel="stylesheet">
<script type="module">
    import ChatbotWidget from 'https://your-domain.github.io/webhook-chatbot/assets/chatbot-widget.js';
    
    document.addEventListener('DOMContentLoaded', function() {
        const chatbot = new ChatbotWidget({
            webhookUrl: 'YOUR_WEBHOOK_URL',
            title: 'Your Bot Name',
            primaryColor: '#007bff',
            // Add other configuration options as needed
        });
    });
</script>

Replace your-domain with your GitHub username and YOUR_WEBHOOK_URL with your n8n webhook URL.

Configuration Options

Basic Options

  • webhookUrl: Your n8n webhook URL
  • title: Chatbot title
  • placeholder: Input placeholder text
  • welcomeMessage: Initial message

Styling Options

  • primaryColor: Main theme color
  • chatBackgroundColor: Background color
  • headerTextColor: Header text color
  • position: 'bottom-right', 'bottom-left', or 'inline'

Custom Branding

  • botIcon: Custom icon URL
  • titleLogo: Logo URL for header
  • showLogoInChat: Show logo in messages
  • footerText: Text in footer

Usage Examples

// Basic usage
const chatbot = new ChatbotWidget({
    webhookUrl: 'https://n8n.example.com/webhook/abc123',
    title: 'Support Bot',
    primaryColor: '#e91e63'
});

// With suggested questions
const chatbot = new ChatbotWidget({
    webhookUrl: 'https://n8n.example.com/webhook/abc123',
    title: 'Support Bot',
    primaryColor: '#4CAF50',
    suggestedQuestions: [
        'What services do you offer?',
        'How can I contact support?',
        'Where are you located?'
    ]
});

// Embedded in a specific container
const chatbot = new ChatbotWidget({
    webhookUrl: 'https://n8n.example.com/webhook/abc123',
    containerId: 'my-chat-container',
    position: 'inline'
});
